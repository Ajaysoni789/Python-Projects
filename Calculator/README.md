Python Calculator
Description

This is a basic command-line calculator written in Python.
It takes two numbers and an arithmetic operator from the user and performs the calculation.

Supported operations:

Addition (+)

Subtraction (-)

Multiplication (*)

Division (/)

The program also handles division by zero.

Features

Beginner-friendly Python code

Uses input() for user interaction

Uses conditional statements (if / elif / else)

Prevents division by zero errors

Runs directly in the terminal

Requirements

Python 3.x installed on your system

To check Python version:

python --version

How to Run the Program

Clone the repository or download the file:

git clone <repository-url>


Navigate to the project folder:

cd simple-python-calculator


Run the program:

python calculator.py

Example Usage
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): /
Result: 2.0

Code Overview

The program asks the user for:

First number

Second number

Arithmetic operator

It checks the operator using if-elif conditions

Performs the corresponding calculation

Displays the result in the terminal
