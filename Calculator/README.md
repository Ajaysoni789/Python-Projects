# Simple Python Calculator

## Description
This is a basic **command-line calculator** written in Python.  
It takes two numbers and an arithmetic operator from the user and performs the calculation.

Supported operations:
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)

The program also handles division by zero.

---

## Features
- Beginner-friendly Python code
- Uses `input()` for user interaction
- Uses conditional statements (`if / elif / else`)
- Prevents division by zero errors
- Runs directly in the terminal

---

## Requirements
- Python 3.x installed on your system

Check Python version:
```bash
python --version
```

---

## How to Run the Program

1. Clone the repository or download the file:
```bash
git clone <repository-url>
```

2. Navigate to the project folder:
```bash
cd simple-python-calculator
```

3. Run the program:
```bash
python calculator.py
```

---

## Example Usage

```
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): /
Result: 2.0
```
